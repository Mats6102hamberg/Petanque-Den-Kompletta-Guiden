📝 Instruktioner för att uppdatera arkivet:

  1. I slutet av varje månad: Kopiera allt innehåll från aktuellt.html
  2. Skapa en ny month-section längst upp i arkivet med aktuell månad
  3. Klistra in artiklarna som archive-article element
  4. Uppdatera statistiken i archive-intro (antal månader, artiklar, intervjuer)
  5. Språkbadges: Använd 🇸🇪 🇬🇧 🇫🇷 🇪🇸 🇩🇪 🇹🇭 för att visa vilket språk artikeln är på
  6. Kategorier: Använd samma kategorier som i aktuellt.html

Tips: Äldre månader kan vara collapsed (class="collapsed") som standard för att hålla sidan överskådlig.
